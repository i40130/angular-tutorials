PIPES

Transformando Datos en las Plantillas

¿Qué son los Pipes en Angular?

Los pipes en Angular son una forma concisa de transformar datos antes de mostrarlos en la plantilla. Permiten formatear fechas, números, cadenas y realizar otras operaciones de transformación de datos sin necesidad de escribir código complejo en el componente.

¿Para qué sirven los Pipes?

  • Formateo de fechas: Mostrar fechas en diferentes formatos (ej: ‘dd/MM/yyyy’, ‘MMMM dd, yyyy’).
  • Formateo de números: Mostrar números con un determinado número de decimales, con separadores de miles, etc.
  • Formateado de cadenas: Convertir a mayúsculas, minúsculas, capitalizar, etc.
  • Creación de pipes personalizados: Para realizar transformaciones específicas que no estén incluidas en los pipes predefinidos.

Tipos de Pipes

  • Pipes puros: Se ejecutan cada vez que cambian las entradas. Son ideales para transformaciones simples y rápidas.
  • Pipes impuros: Se ejecutan en cada ciclo de detección de cambios. Se utilizan para transformaciones más complejas que requieren acceder a servicios o realizar operaciones costosas.

Pipes Predefinidos

  • DatePipe: Formatea fechas.
  • NumberPipe: Formatea números.
  • UpperCasePipe: Convierte una cadena a mayúsculas.
  • LowerCasePipe: Convierte una cadena a minúsculas.
  • CurrencyPipe: Formatea valores monetarios.
  • PercentPipe: Formatea valores porcentuales.
  • DecimalPipe: Formatea números decimales.

Ejemplo de uso

HTML

<p>Hoy es {{ today | date:'fullDate' }}</p>
<p>El precio es: {{ price | currency:'USD' }}</p>

Creación de un Pipe Personalizado:

TypeScript

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'truncate'
})
export class TruncatePipe implements PipeTransform {
  transform(value: string, limit: number = 25): string {
    return value.length > limit ? `${value.substring(0, limit)}...` : value;
  }
}

Uso del Pipe Personalizado:

HTML

<p>{{ longText | truncate:10 }}</p>

Ventajas de usar Pipes

  • Legibilidad: Mejoran la legibilidad de las plantillas al separar la lógica de presentación de la lógica del componente.
  • Reutilización: Los pipes pueden ser reutilizados en diferentes componentes.
  • Mantenimiento: Facilitan el mantenimiento del código al centralizar la lógica de formateo.

Consideraciones:

  • Rendimiento: Los pipes puros son más eficientes que los impuros.
  • Complejidad: Para transformaciones muy complejas, puede ser más adecuado crear un servicio.
  • Pureza: Asegúrate de que un pipe sea puro si no necesita acceder a servicios o realizar operaciones con efectos secundarios.

Los pipes son una gran herramienta en Angular para transformar datos y mejorar la presentación de la información en las plantillas. Al utilizar pipes de forma efectiva, puedes crear aplicaciones más limpias, mantenibles y con una mejor experiencia de usuario.
