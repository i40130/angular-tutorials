DEFINICIÓN DE COMPONENTES

En Angular, los componentes son la base fundamental para construir interfaces de usuario. Son bloques de construcción reutilizables que encapsulan una porción de la interfaz, su lógica y su estilo.

Componentes en Angular

¿Cómo se define un componente?

Para definir un componente en Angular, se utiliza el decorador @Component junto con una clase TypeScript. Este decorador proporciona metadatos sobre el componente, como su selector, plantilla y estilos.

Estructura básica de un componente

TypeScript

import { Component } from '@angular/core';

@Component({
  selector: 'app-mi-componente',
  templateUrl: './mi-componente.component.html',
  styleUrls: ['./mi-componente.component.css']
})
export class MiComponenteComponent {
  // Propiedades y métodos del componente
}

Explicación de cada parte:

  • @Component: Este decorador marca la clase como un componente y recibe un objeto de configuración.

    • selector: Un string que indica el nombre del elemento HTML que representará al componente en la plantilla. En este caso, <app-mi-componente>.
    • templateUrl: La ruta al archivo HTML que contiene la plantilla del componente.
    • styleUrls: Un array de rutas a los archivos CSS que contienen los estilos del componente.

  • Clase TypeScript:

    • Contiene la lógica del componente, como propiedades, métodos y eventos.
    • Las propiedades se pueden utilizar para almacenar datos que se mostrarán en la plantilla.
    • Los métodos pueden realizar acciones en respuesta a eventos del usuario o cambios en los datos.

Ejemplo de plantilla (mi-componente.component.html):

HTML

<p>Hola desde mi componente!</p>
<button (click)="saludar()">Saludar</button>

Ejemplo de estilo (mi-componente.component.css):

CSS

p {
  color: blue;
}

¿Cómo se utiliza un componente?

Una vez definido un componente, se puede utilizar en otras partes de la aplicación incluyéndolo en la plantilla de otro componente o en el index.html principal. Por ejemplo:

HTML

<app-mi-componente></app-mi-componente>

Los componentes en Angular son unidades independientes y reutilizables que encapsulan una porción de la interfaz de usuario. Al definir un componente, se especifica su selector, plantilla y estilos. Estos componentes se pueden anidar y combinar para crear interfaces de usuario complejas y dinámicas.
