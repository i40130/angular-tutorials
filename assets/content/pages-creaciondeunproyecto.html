CREACION DE UN PROYECTO

Creación de un Proyecto en Angular: Una Guía Paso a Paso

Angular es un framework de JavaScript de alto nivel, respaldado por Google, que se utiliza para crear aplicaciones web de una sola página (SPA). Para comenzar un nuevo proyecto en Angular, necesitarás la Angular CLI (Command Line Interface).

1. Instalación de Angular CLI

Antes de comenzar, asegúrate de tener Node.js y npm (o yarn) instalados en tu sistema. Luego, abre tu terminal o línea de comandos y ejecuta el siguiente comando:

Bash

npm install -g @angular/cli

Esto instalará globalmente la Angular CLI.

2. Creación de un Nuevo Proyecto

Una vez instalada la Angular CLI, puedes crear un nuevo proyecto de Angular ejecutando el siguiente comando:

Bash

ng new mi-proyecto

Reemplaza mi-proyecto con el nombre que deseas asignarle a tu proyecto. Se te pedirá que selecciones algunas opciones, como el estilo (CSS, Sass, Less, Stylus) y si deseas incluir Angular Routing.

3. Estructura del Proyecto

Al finalizar la creación del proyecto, tendrás una estructura de directorios similar a esta: 

mi-proyecto/
  e2e/
  node_modules/
  src/
    app/
      app.component.css
      app.component.html
      app.component.spec.ts
      app.component.ts
      app.module.ts
    assets/
    environments/
    index.html
    main.ts
    styles.css
    tsconfig.app.json
    tsconfig.spec.json
  .angular.cli.json
  package.json
  README.md
  tsconfig.json
  • src: Contiene el código fuente de tu aplicación.
  • app: Aquí se encuentran los componentes principales de tu aplicación.
  • assets: Para almacenar archivos estáticos como imágenes, fuentes, etc.
  • environments: Para configurar diferentes entornos (desarrollo, producción).

4. Ejecutar la Aplicación

Para ejecutar tu aplicación localmente, navega al directorio de tu proyecto y ejecuta:

Bash

cd mi-proyecto
ng serve

Esto iniciará un servidor de desarrollo local y abrirá tu aplicación en el navegador por defecto (generalmente en http://localhost:4200/).

5. Estructura Básica de un Componente

Un componente en Angular tiene tres partes principales:

  • Template (HTML): Define la vista del componente.
  • Style (CSS): Estiliza el componente.
  • Logic (TypeScript): Contiene la lógica del componente.

TypeScript

// app.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'mi-proyecto';
}

HTML

<h1>{{ title }}</h1>

6. Crear Nuevos Componentes

Para crear nuevos componentes, utiliza el siguiente comando:

Bash

ng generate component my-component

Esto creará un nuevo componente llamado my-component con sus archivos correspondientes.

Consideraciones Adicionales

  • Angular CLI: Te permite generar muchos otros elementos como servicios, directivas, pipes, etc.
  • Módulos: Organizan tu aplicación en módulos, mejorando la modularidad y la mantenibilidad.
  • Routing: Permite crear una aplicación de una sola página con múltiples vistas.
  • Reactive Forms: Un poderoso mecanismo para manejar formularios en Angular.
  • HTTP Client: Para realizar solicitudes HTTP a servidores externos.
