MÓDULOS

Los módulos en Angular son una pieza fundamental para la organización y estructuración de tus aplicaciones. Son como contenedores que agrupan componentes, servicios, directivas y otros artefactos relacionados, creando bloques de funcionalidad cohesivos.

¿Qué son los módulos en Angular?

Un módulo en Angular es una clase TypeScript decorada con @NgModule que define un contexto para un conjunto de funcionalidades relacionadas. Es como un mini-aplicación dentro de una aplicación más grande.

¿Para qué sirven los módulos?

  • Organización: Dividen la aplicación en partes más pequeñas y manejables.
  • Reutilización: Permiten crear módulos reutilizables que pueden ser importados en diferentes partes de la aplicación.
  • Carga perezosa: Los módulos pueden cargarse de forma perezosa, lo que mejora el rendimiento de la aplicación, especialmente en aplicaciones grandes.
  • Enrutamiento: Los módulos se utilizan para definir rutas y navegar entre diferentes partes de la aplicación.

Estructura básica de un módulo:

TypeScript

import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { MiComponente } from './mi-componente';

@NgModule({
  declarations: [MiComponente],
  imports: [CommonModule],
  exports: [MiComponente]})
export class MiModuloModule { }
  • declarations: Lista de componentes, directivas y pipes declarados en este módulo.
  • imports: Lista de otros módulos que este módulo necesita importar.
  • exports: Lista de declaraciones que este módulo exporta para que puedan ser utilizados por otros módulos.

¿Cómo funcionan los módulos?

  • Módulo raíz: Toda aplicación Angular tiene un módulo raíz, que es el punto de entrada de la aplicación.
  • Módulos hijos: Los módulos hijos pueden importar otros módulos y exportar sus componentes para que sean utilizados por otros módulos.
  • Enrutamiento: Los módulos se utilizan para definir rutas y cargar diferentes módulos de forma perezosa.

Ejemplo de un módulo con enrutamiento

TypeScript

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';

const routes: Routes = [{ path: 'productos', loadChildren: () => import('./productos/productos.module').then(m => m.ProductosModule)}];

@NgModule({
  imports: [RouterModule.forChild(routes)],
  exports: [RouterModule]})
export class MiModuloRoutingModule { }

En este ejemplo, el módulo MiModuloRoutingModule define una ruta para cargar el módulo ProductosModule de forma perezosa.
