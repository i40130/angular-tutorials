INSTALACION DE ANGULAR

Angular es un framework de JavaScript muy popular para el desarrollo de aplicaciones web. Para comenzar a trabajar con él, lo primero que necesitas es instalarlo en tu entorno de desarrollo.

Requisitos previos

  • Node.js y npm (o yarn): Estos son fundamentales para la gestión de paquetes en JavaScript. Puedes descargarlos desde https://nodejs.org/.

Pasos para la instalación

  1. Verifica las versiones:

    • Abre tu terminal o línea de comandos.

    • Ejecuta los siguientes comandos para verificar las versiones instaladas de Node.js y npm:
      Bash 

      node -v
npm -v

    • Si aún no los tienes instalados, sigue las instrucciones de la página oficial de Node.js.

  2. Instala Angular CLI:

    • Angular CLI (Command Line Interface) es una herramienta que simplifica la creación y gestión de proyectos Angular.
    • Ejecuta el siguiente comando en tu terminal.
      Bash 
      npm install -g @angular/cli

  3. Verifica la instalación:

    • Ejecuta el siguiente comando para verificar si Angular CLI se instaló correctamente:

      Bash

      ng --version

    • Deberías ver la versión instalada de Angular CLI.

  4. Crea un nuevo proyecto:

    • Navega al directorio donde deseas crear tu proyecto.

    • Ejecuta el siguiente comando para crear un nuevo proyecto Angular:

      Bash

      ng new mi-proyecto

      Reemplaza mi-proyecto con el nombre que quieras darle a tu proyecto.

  5. Ejecuta la aplicación:

    • Una vez creado el proyecto, navega a su directorio:

      Bash

      cd mi-proyecto

    • Inicia el servidor de desarrollo:

      Bash

      ng serve

    • Abre tu navegador y ve a http://localhost:4200/. Deberías ver la página de bienvenida de tu nueva aplicación Angular.

Explicación de los comandos

  • npm install -g @angular/cli: Instala Angular CLI globalmente, lo que te permite usar el comando ng desde cualquier directorio.
  • ng new mi-proyecto: Crea un nuevo proyecto Angular con el nombre especificado. Genera toda la estructura de archivos y configuraciones necesarias.
  • ng serve: Inicia un servidor de desarrollo local y compila tu aplicación en tiempo real.

Personalización

  • Opciones adicionales:
    • ng new mi-proyecto --style=scss: Utiliza Sass como preprocesador de estilos.
    • ng new mi-proyecto --routing: Genera un módulo de enrutamiento.
    • Para ver todas las opciones disponibles, ejecuta ng new --help.
  • Configuración:
    • El archivo angular.json contiene la configuración global de tu proyecto.
    • El archivo tsconfig.json configura el compilador TypeScript.
